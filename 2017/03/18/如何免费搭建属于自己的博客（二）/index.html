如何免费搭建属于自己的博客（二）

发表于 2017-03-18 | 分类于 总结 | | 阅读次数

0. 写在前面

使用hexo+Github搭建完博客后，界面比较简陋，功能几乎没有，那如何美化自己的博客，并增加自己想要的功能（比如评论、统计访客数量等）呢？这个时候可以读一下hexo官方文档。可以通过使用主题、增加插件等方式将简陋的博客改造成自己想要的样子。博主使用的是第三方的主题-next进行的美化，感兴趣的朋友可以试试。下面就来介绍如何使用next主题来增加自己想要的功能。

1. 如何安装next

next官方网站上有详细介绍，这里就不多说了，安装方式如下：

在终端窗口下，定位到 Hexo 站点目录下。使用 Git checkout 代码：

cd your-hexo-site
git clone https://github.com/iissnan/hexo-theme-next themes/next

这时，可以看到在themes文件夹下多了next的文件夹。

2. 如何修改为next主题

首先明确什么是站点配置文件、主题配置文件。

站点配置文件：hexo文件夹下的’_config.yml’文件

主题配置文件：next文件夹下的’_config.yml’文件

打开站点配置文件，找到 theme 字段，并将其值更改为 next。

theme: next

接下来验证主题是否启用。键入以下命令：

hexo clean
hexo s --debug #启动调试模式

出现

INFO  Hexo is running at http://0.0.0.0:4000/. Press Ctrl+C to stop.

表明可以使用浏览器访问http://0.0.0.0:4000/，检查站点是否正确运行。至此，我们将主题修改成next。

美化过程，next官方文档已有详细介绍，如何修改为博主的界面呢？下面介绍主要的修改过程，对其他功能感兴趣可以给我留言。

3. 如何增加自己想要的功能

3.1 更改头像

打开主题配置文件：

# Sidebar Avatar
# in theme directory(source/images): /images/avatar.jpg
# in site  directory(source/uploads): /uploads/avatar.jpg
avatar: /images/touxiang.jpg

3.2 更改网页小图标

打开主题配置文件：

# Put your favicon.ico into `hexo-site/source/` directory.
favicon: /favicon.ico

3.3 关闭’由hexo强力驱动’

打开主题配置文件：

# Footer `powered-by` and `theme-info` copyright
copyright: false

3.4 增加赞赏功能

打开主题配置文件，添加：

reward_comment: 坚持原创技术分享，您的支持将鼓励我继续创作！
wechatpay: /images/weichat.png

4. 增加第三方服务

4.1 增加多说评论

登录多说建立站点（详细见next官方文档）。
打开主题配置文件：

# Duoshuo ShortName
duoshuo_shortname: xxlee2009

4.2 统计访客数量

使用不蒜子进行统计。

打开themes/你的主题/layout/_partial/footer.swig添加以下脚本，要添加在开始位置：

<script async src="//dn-lbstatics.qbox.me/busuanzi/2.3/busuanzi.pure.mini.js">
</script>

若添加一首歌，则点击歌曲页面找到“生成外链播放器”,点击。

复制HTML代码

  • 如果选择放到主页的话，就放到_layout.swig文件中去就行了

博客歌单会随着你网易云音乐歌单更新而更新。

5. 预览与发布

添加完成后，要在本地预览一下：

hexo clean
hexo g
hexo s

打开http://localhost:4000/就可以看到。

若本地预览没问题，就发布到Github上，以后直接登录你的博客就可以看到经过定制后的页面。

hexo deploy
